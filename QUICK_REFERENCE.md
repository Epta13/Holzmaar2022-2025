## Known Issues & Workarounds

| Issue | Impact | Workaround |
|-------|--------|-----------|
| Timestamps in filenames | Non-reproducible outputs | Use fixed names or seed |
| Inconsistent cluster counts (k=3 vs k=5) | Interpretation conflicts | Document reasoning |
| 50% zero filtering threshold (nb 07) | Rare species removal | Justify or adjust |
| Merge row expansion (nb 08) | Data multiplication | Use join instead |
| Age model column inconsistency | Silent misalignment | Use consistent naming |
| Extrapolation (rule=2) | Unreliable edge data | Document assumptions |
